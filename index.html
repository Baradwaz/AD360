Key Modules in AD360

User Provisioning & Deprovisioning

  • Automate onboarding across AD, Exchange, Microsoft 365, and Google Workspace
  • Role-based templates, CSV imports, and HRMS integrations
  • Bulk user management with zero-touch automation

Multi-Factor Authentication (MFA)

  • MFA for AD logins, VPN, RD Web, OWA, endpoints
  • 20+ authenticators including FIDO2, TOTP, and biometrics
  • Enforce MFA based on user group, IP, time, or device

Help Desk Delegation

  • Granular, OU-based task delegation to non-admin users
  • Audit logs of delegated actions and role-based controls
  • Web-based AD management interface for help desk teams

Active Directory Change Auditing

  • Track changes to users, groups, GPOs, logons, file shares
  • Real-time alerts for privileged activity or policy violations
  • Preconfigured compliance-ready reports (GDPR, HIPAA, SOX)

Reports & Compliance

  • 300+ pre-built reports for auditing, logons, inactive users
  • Export to PDF/CSV, schedule reports, automate delivery
  • Track privilege escalations and stale accounts

Identity Governance & Review

  • Periodic access review campaigns for AD, M365, apps
  • Certify entitlements, detect toxic combinations
  • Delegate reviewers and maintain audit trails

Group Management & Cleanups

  • Bulk manage distribution & security groups
  • Identify nested, unused, or empty groups
  • Automated clean-up based on inactivity rules

Integrations & APIs

  • Connect with HR systems (Workday, BambooHR, SAP)
  • REST APIs for external IAM workflows
  • SSO, SIEM, ticketing, and IAM tools integration

Self-Service Portal

  • End users reset passwords, unlock accounts securely
  • AD attribute updates via browser or mobile app
  • Multi-factor validation before changes apply
